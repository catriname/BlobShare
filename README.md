# BlobShare

A simple UI for showing, searching and sharing Azure Blobs.  Retrieves blobs from a specified container and creates both a search and tree view of the blobs.

Uses:
* JQuery
* zTree
* Azure Storage
* Azure Blobs
* .NET Core 2.1
